# EnderEggRespawn

Плагин для Minecraft (Paper/Spigot), который позволяет яйцу Дракона выпадать при каждом убийстве Эндер-дракона, а не только в первый раз.

## Особенности

- Яйцо Дракона появляется после каждого убийства Эндер-дракона
- Возможность настройки метода появления яйца (как блок или как предмет)
- Поддержка кастомных координат для появления яйца
- Настраиваемое сообщение об оповещении
- Возможность отключения оповещений
- Настраиваемая задержка перед появлением яйца

## Установка

1. Скачайте плагин в формате .jar
2. Поместите файл в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Настройте плагин в файле `config.yml`

## Настройка

После первого запуска плагин создаст файл `config.yml` в папке `plugins/EnderEggRespawn/`:

```yaml
# Конфигурационный файл плагина EnderEggRespawn
# Здесь вы можете настроить поведение яйца Дракона после убийства Эндер-дракона

# Метод появления яйца: BLOCK - как блок, ITEM - как предмет
# BLOCK: Яйцо появляется как размещённый блок на заданных координатах
# ITEM: Яйцо появляется как предмет, который можно подобрать
spawn-method: BLOCK

# Настройки кастомных координат для появления яйца
custom-coords:
  # Включить использование кастомных координат
 # Если false, яйцо будет появляться в центре портала (над бедроком)
  enabled: false
  # Координаты X для появления яйца (работает, если enabled: true)
  x: 0
  # Координаты Y для появления яйца (работает, если enabled: true)
  y: 0
  # Координаты Z для появления яйца (работает, если enabled: true)
  z: 0

# Сообщение, которое будет отправлено всем игрокам при появлении яйца
# Поддерживает цветовые коды Minecraft (например, §a - зелёный, §c - красный)
announcement-message: "§aЯйцо Дракона появилось после убийства Эндер-дракона!"

# Включить/выключить оповещение о появлении яйца
# true - сообщение будет отправлено всем игрокам
# false - сообщение не будет отправлено
enable-broadcast: true

# Задержка в тиках перед спавном яйца (20 тиков = 1 секунда)
# Эта задержка нужна для того, чтобы портал дракона успел восстановиться
# после смерти дракона, особенно если используется метод BLOCK
respawn-delay-ticks: 20

# Дополнительные настройки спавна
spawn-settings:
  # Проверять, что яйцо не появится внутри другого блока
  # Если true, плагин будет искать ближайшее свободное место для спавна яйца
  check-for-free-space: true
  # Максимальная высота, на которой будет искаться место для спавна яйца
  # (если текущая позиция занята другим блоком)
  max-search-height: 256
  # Минимальная высота, на которой будет искаться место для спавна яйца
  min-search-height: 0
  # Пытаться размещать яйцо на вершине портала (если custom-coords.enabled: false)
  # Если true, плагин будет пытаться найти верхнюю часть портала дракона
  use-portal-top: true

# Параметр firstKill - указывает, было ли первое убийство дракона
# Если false (по умолчанию), яйцо не будет спавниться от плагина при первом убийстве
# После первого убийства значение становится true, и яйцо будет спавниться при последующих убийствах
firstKill: false
```

## Команды

- `/eer reload` или `/endereggrespawn reload` - Перезагрузить конфигурацию плагина

## Разрешения

- `endereggrespawn.reload` - Позволяет использовать команду перезагрузки конфигурации (по умолчанию: op)

## Совместимость
